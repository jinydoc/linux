SSH 접속하기

ssh를 이용하여 리눅스 서버에 접속을 합니다.

01.ssh 설치하기

ssh를 통하여 서버에 접속을 하기 위해서는 별도의 ssh 서비스 서버 프로그램을 설치해 주셔야 합니다. ssh 서버를 구축하기 위해서는 2단계 작업이 필요 합니다.

  • 방화벽 허용하기
  • ssh 패키지 설치하기
  • root로 접속 허용하기

02.ssh로 터미널 접속하기

ssh로 터미널 접속을 하기 위해서는 전용 프로그램이 필요 합니다.

  • putty
  • MobaXterm
  • WSL설치
  • 명령어
