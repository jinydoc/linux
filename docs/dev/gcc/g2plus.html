WSL g++ 설치하기

hojin@DESKTOP-11LMH3B:~$ g++ --version
Command 'g++' not found, but can be installed with:
sudo apt install g++

g++을 설치합니다.

sudo apt -y install g++

설치가 완료된후에 다시 g++ 명령을 입력해 봅니다.

hojin@DESKTOP-11LMH3B:~$ g++ --version
