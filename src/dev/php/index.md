# php 설치
PHP는 소스를 컴파일 하여 설치하는 방법과 패티지를 이용하는 설치하는 방법이 있습니다.

패키지의 경우 운영체제에 최적화 되어 있다는 점과, 쉽게 설치를 할 수 있는 점에서 많이 선호 합니다.


# php 연동설치
---

```console
sudo apt -y install php
```
