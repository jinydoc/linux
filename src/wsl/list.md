# 윈도우에 설치된 wsl 확인하기
---
`윈도우10`의 WSL 환경은 복수의 다양한 리눅스 배포본을 설치할 수 있습니다. 

## wsl 명령어
설치된 배포본의 목록은 `wsl` 명령과 `-l` 옵션을 통하여 확인할 수 있습니다.  

윈도우 `CMD` 터미널을 실행한 후에 다음과 같이 명령을 입력합니다.

```console
C:\> wsl -l
Linux 배포용 Windows 하위 시스템:
Ubuntu-20.04(기본값)
```

하나의 `Ubuntu-20.04` 배포톤이 설치되어 있는 것을 확인할 수 있습니다.
