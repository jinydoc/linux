# PHP
---
php는 웹 응용프로그램을 개발하기 위한 스크립트 언어 입니다. 리눅스에 php를 설치하여 실행하는 방법에 대해서 알아 봅니다. php를 설치하여 사용하기 위해서는 먼저 apache 또는 nginx와 같은 웹서버가 미리 설치되어 있어야 합니다.

<br>
